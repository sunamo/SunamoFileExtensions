# NuGet Alternatives to SunamoFileExtensions

This document lists popular NuGet packages that provide similar functionality to SunamoFileExtensions.

## Overview

File extension utilities

## Alternative Packages

### System.IO.Path
- **NuGet**: System.Runtime
- **Purpose**: Built-in path operations
- **Key Features**: GetExtension, ChangeExtension, file name parsing

### MimeMapping
- **NuGet**: MimeMapping
- **Purpose**: Extension to MIME type mapping
- **Key Features**: File type detection from extensions

### HeyRed.Mime
- **NuGet**: HeyRed.Mime
- **Purpose**: MIME type detection
- **Key Features**: Extension lookup, content-based detection

## Comparison Notes

System.IO.Path for basic extension operations. MimeMapping for MIME types.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **System.IO.Path**: Built-in path operations
- **MimeMapping**: Extension to MIME type mapping
- **HeyRed.Mime**: MIME type detection
